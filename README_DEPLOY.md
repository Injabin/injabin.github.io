# 🚀 How to Deploy to GitHub Pages (FREE)

## Step 1 — Create the repo
1. Go to github.com → New Repository
2. Name it EXACTLY: `Injabin.github.io`
3. Set to Public
4. Click "Create repository"

## Step 2 — Upload files
Upload these files to the repo:
- `index.html`
- `avatar.png`

## Step 3 — Enable GitHub Pages
1. Go to repo Settings → Pages
2. Source: "Deploy from a branch"
3. Branch: `main` → `/root`
4. Click Save

## Step 4 — Live in ~1 minute!
Your site will be live at: **https://Injabin.github.io**

---
That's it! Free hosting, custom domain optional.
